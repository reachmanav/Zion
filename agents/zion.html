ZION

The Last Human City — WhatsApp AI Bot

The central intelligence of The Matrix. Zion receives messages from Neo and friends via WhatsApp, processes them through a sophisticated routing pipeline, and responds using AI, web search, real-time APIs, and 12+ sub-constructs.

Internal Architecture

How a WhatsApp message flows through Zion

1
MESSAGE ARRIVES
User sends @zion weather in Pune on WhatsApp
WhatsApp Bridge (Go) → SQLite messages.db → Zion polls every 2s
▼
2
PARSE & CLASSIFY
Strip @zion tag, detect media type (text/image/voice/document), check permissions
Group approval gate • Owner check • Context window • Follow-up detection
▼
3
SMART ROUTER
Priority-ordered handler chain — first match wins
🔺 @trinity → Trinity Queue
🖼️ Image → Gemini Vision
🎤 Voice → STT → Text pipeline
📄 Document → Summary/Review
🔗 Link → Auto-summarize
🏋️ Strava → Fitness data
🎯 Trivia → Game engine
📅 Reminder → Calendar
📝 Summarize → LLM summary
✉️ Send → Message relay
👥 Contact → Google Contacts
🌤️ Weather → Open-Meteo API (direct, no LLM)
🤖 Default → Groq LLM (Llama 3.3 70B)
▼
4
PROCESS & ENRICH
Handler fetches data, calls APIs, generates response
Web search (DDG + Google News) • Weather API • LLM with chat history • Image analysis
▼
5
REPLY
Formatted response sent back via WhatsApp Bridge
send_reply() → Bridge API :8080 → WhatsApp → User's phone

Interactive Features

Real-time capabilities beyond daily briefs

🌤️
Weather

Real-time weather from Open-Meteo API. Direct data — no LLM hallucination. wttr.in fallback.

🖼️
Image Vision

Send a photo → Gemini Flash analyzes it → describes content, reads text, answers questions.

🎤
Voice Notes

Send a voice note → Gemini/Whisper transcribes → processes as text command.

🔗
Link Summary

Drop a URL → Zion fetches and summarizes the article in 3-4 bullet points.

📄
Document Analysis

Send a PDF/document → extracts text → summarizes or reviews code.

🎯
Trivia Games

Multi-player trivia in group chats. Scores, leaderboards, topic selection.

👥
Contacts

Search Google Contacts by name. Smart relevance scoring, clean formatting.

📅
Calendar

Create Google Calendar events and get daily reminders via WhatsApp.

Tech Stack

Runtime Python 3 — systemd service (always on)
Primary LLM Groq — Llama 3.3 70B (free tier)
Vision / STT Gemini Flash (primary) + Groq Whisper (fallback)
Search Weather: Open-Meteo + wttr.in | News: Google RSS | Web: DuckDuckGo
Messaging WhatsApp Bridge (Go, whatsmeow) on port 8080
Database SQLite — messages.db, zion_data.db, contacts.db, search_cache.db
Scraping Playwright (headless Chromium) for Google Flights
Cost $0 — all free tier APIs + Oracle Cloud Free Tier
