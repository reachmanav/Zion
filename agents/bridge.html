WHATSAPP BRIDGE

The Gateway — Go Service on Port 8080

Every WhatsApp message enters and leaves through this bridge. Written in Go, it connects to WhatsApp servers, stores messages in SQLite, and provides a REST API for all agents.

Media/tmp/whatsapp/ — auto-downloaded files
Servicewhatsapp-bridge.service (systemd)
