TECH STACK

FRAMEWORK
NVIDIA NeMo Guardrails (open source, Apache 2.0)
CONFIG LANG
Colang — define topical rules, refusal flows, safety filters
RAILS
Input rails (intent check) + Output rails (PII/secret leak check)
INTEGRATION
Python middleware in zion.py — every message wrapped before reaching Trinity / CLI / Zion handlers
BLOCKLIST
credentials, financial data, destructive cmds (rm/delete/drop), prompt injection, /etc/* /home/* file access, bulk exports
LATENCY
~200-400 ms per check (3-rail setup), runs locally on VM
COST
FREE — runs on the existing Ampere A1 VM, no API calls
ALERT
Every BLOCK pings Neo via WhatsApp: "[Nemo] Blocked: {sender} tried {request}"
